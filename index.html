Project

Leaflet Map

Interactive Map Application with Leaflet
Project Overview
This project is an interactive map application developed using Leaflet.js, an open-source JavaScript library for mobile-friendly interactive maps. The application integrates various geospatial functionalities including dynamic map layers, geolocation, and marker clustering, providing an intuitive and user-friendly interface for visualizing geographical data.

Key Features :

Dynamic Basemap Layers:
Users can switch between different basemaps such as Streets and Satellite views, powered by Esri's ArcGIS Online services.

Geolocation:
The application can locate the user's current position, centering the map and placing a marker to indicate their location.

Marker Clustering:
Utilizes the Leaflet.markercluster plugin to efficiently display a large number of markers, enhancing performance and user experience.

Custom Markers:
Markers are customized using the Leaflet.extra-markers plugin, enabling distinctive icons and colors for different types of locations.

Layer Control:
Users can toggle between different overlay layers, such as city markers, providing flexibility in data visualization.

Interactive Popups:
Markers are equipped with interactive popups displaying detailed information about each location, including name and population
